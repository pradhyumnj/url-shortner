URL shortner Project

To use this app, enter a valid link as input and click on shorten.
The output is a shortned link, but it might not appear as so because the name of my website is long. If it were short, somthing like bit.ly, then the url will look shorter.
Paste this new url in your browser, you'll be redirected to the link you orignally entered.
